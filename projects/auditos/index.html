Overview

A practical desktop project built around privacy review and clearer risk reporting.

AuditOS is a desktop application I built to explore local-first privacy auditing, browser configuration review, and user-friendly technical reporting.

The project combines my interests in technical support, security awareness, troubleshooting, software design, and practical risk reduction.

Project focus

What the project is designed to support

Local-first auditing Browser review Privacy posture Readable reporting

AuditOS is meant to turn scattered browser privacy and security signals into something more understandable, reviewable, and actionable.

The Problem

Browser setup risk is real, but most people never get a clear read on it.

Most users do not have an easy way to understand what their browser setup may expose, how extensions or settings may affect privacy, or where common security gaps may exist. AuditOS was created as a practical tool to make that kind of review more understandable.

What I Built

Core pieces of the application.

AuditOS is structured to keep the experience clear for users while leaving room for the audit system to expand over time.

Build area

Python desktop application

Built as a desktop tool to keep the workflow local, direct, and easier for users to trust.

Build area

Local-first audit workflow

Designed around reviewing information on the user’s machine instead of relying on a remote service model.

Build area

Browser privacy/security review

Focused on the kinds of browser settings, extensions, and configurations that shape privacy posture and exposure.

Build area

Modular audit structure

Organized so additional checks and audit categories can be added without forcing a full rewrite later.

Build area

User-focused reporting

Shaped to present technical findings in a way that is still readable for people who are not deeply technical.

Build area

GitHub-based open-source project structure

Set up publicly so the project can support releases, testing, documentation, and outside contribution over time.
